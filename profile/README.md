## What BoTorch Brings to Model-Based Optimization

Download BoTorch tutorial to learn practical Bayesian model-based search with a modern PyTorch library. Explore setup, examples, custom models, acquisition strategies, and links to BoTorch documentation so researchers and developers can build reliable experiments faster today.

BoTorch is an open source PyTorch library for Bayesian optimization, helping teams design experiments, tune models, and optimize complex systems.

Built for researchers, engineers, and machine learning teams, BoTorch focuses on flexible Bayesian optimization workflows that integrate with PyTorch, GPyTorch, and Ax. A strong BoTorch tutorial can help new users understand probabilistic modeling, while BoTorch documentation explains how to compose models, objectives, constraints, and candidate generation pipelines for real experiments.

The project is useful when objective evaluations are expensive, noisy, constrained, or multi-output. With BoTorch examples, teams can prototype custom optimization loops, compare acquisition strategies, and adapt BoTorch Python code for laboratory experiments, simulation studies, hyperparameter tuning, materials discovery, robotics, and product design systems.
